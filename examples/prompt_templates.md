# Prompt Templates

## Canonical style prompt

```text
{subject}, in the {style_title} aesthetic ({era}).
Summary intent: {summary}.
Visual cues: {visual_cues}.
Shape language: {shape_language}.
Pattern and symbols: {pattern_systems}; {symbol_systems}.
Materials, architecture, and fashion cues: {materials}; {architecture}; {fashion}.
Composition and lighting: {composition_rules}; {lighting_atmosphere}.
Typography/graphics overlays: {typography_graphics}.
Palette emphasis: {color_palette}.
Render style: high-detail concept art, coherent diegetic design.
Avoid: {negative_tokens}.
```

## Controlled two-style blend

```text
{subject}. Hybrid retrofuturism: 70% {style_a_title} + 30% {style_b_title}.
Use base composition, materials, and lighting from style A.
Use only motif accents and typography hints from style B.
Keep symbols culturally coherent and avoid chronology conflicts.
```

## Regional variant prompt

```text
{subject} in {style_title}, specifically the {regional_variant} variant.
Historical grounding: {historical_context}.
Include canonical symbols: {symbol_systems}.
Focus subjects: {common_subjects}.
Avoid generalized "global sci-fi" cues and keep regional design DNA explicit.
```

## Contemporary reinterpretation prompt

```text
Contemporary reinterpretation of {style_title} for {application_domain}.
Preserve original shape language and symbol logic.
Translate materials into modern production equivalents.
Keep visual continuity with {historical_context} while integrating {modern_reinterpretation}.
```

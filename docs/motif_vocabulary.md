# Motif and Symbol Vocabulary

## Core motif families

### Motion and progress motifs
- Streamlines
- Orbit arcs
- Speed chevrons
- Launch trajectories

### Industry and control motifs
- Gear rosettes
- Gauge faces
- Wiring bundles
- Panel labels and ID plates

### Civic-ideological motifs
- Emblems and badges
- Monumental reliefs
- Mission patches
- Propaganda poster diagonals

### Domestic-consumer motifs
- Starbursts
- Boomerang inlays
- Product hero pedestals
- Gloss gradients

## Motif compatibility map

- **Strong pairings**
  - Starburst + boomerang (atompunk)
  - Warning stripes + instrument arrays (space-race brutalism/cassette)
  - Mission patch + modular panel IDs (cassette/space-race)
  - Orbit arcs + constructivist diagonals (Soviet cosmist)

- **Risky pairings (use deliberately)**
  - Victorian filigree + Y2K chrome blobs
  - Pulp raygun iconography + hard NASA realism
  - Deco monumental relief + playful atompunk suburban palette

## Symbol systems checklist

When adding symbols to a new style entry, define:

1. Primary symbols (2-5)
2. Where symbols appear (uniform, architecture, interface, print)
3. Symbol function (state identity, safety, ritual, commercial)
4. Level of abstraction (literal, emblematic, procedural)

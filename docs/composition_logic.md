# Composition and Scene Logic

## Canonical composition templates

### 1) Monument + human scale contrast
- Use for: dieselpunk, Soviet cosmist, space-race brutalism.
- Setup: 70% structural mass, 30% human/event action.
- Camera: low or mid-low angle with vertical convergence.

### 2) Interface enclosure frame
- Use for: cassette futurism, retro anime mecha.
- Setup: foreground monitor/console frame + midground operator + distant objective view.
- Camera: shoulder or over-console shot.

### 3) Productized optimistic vignette
- Use for: atompunk, Y2K retrofuture pop.
- Setup: one iconic object against clean environment with controlled accent motifs.
- Camera: mostly eye-level, ad-like framing.

### 4) Heroic pulp diagonal
- Use for: raygun gothic.
- Setup: strong diagonal action vector with exaggerated silhouette readability.
- Camera: dynamic perspective distortion accepted.

## Composition anti-patterns

- Mixing equal-weight motifs from >3 style families in one frame.
- Perfectly symmetric layouts when style requires dynamic narrative energy.
- Uniform texture noise over every surface (removes hierarchy).
- UI overlays that use modern flat-app conventions in analog-era styles.

## Scene grammar by distance

- **Wide shot:** Establish architecture + atmosphere + transportation scale.
- **Mid shot:** Show interaction between people and systems.
- **Close shot:** Reveal material logic, labels, fasteners, wear, and motif details.

Use all three distances for cohesive style worldbuilding.

## How to use this repository

1. Use `data/retrofuturism_entries.json` as the canonical structured dataset.
2. Use the docs above to disambiguate adjacent styles and prevent genre collapse.
3. When adding a new entry, update both the JSON record and at least one relevant guide in `docs/`.
4. Prefer concrete, historically grounded descriptors over broad sci-fi shorthand.

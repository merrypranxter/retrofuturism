# Visual Traits Matrix

## Silhouette signatures by style

| Style | Dominant silhouette | Secondary forms | Avoid |
|---|---|---|---|
| Victorian steampunk | Cylinders, pressure vessels, towers | Trusses, ornate brackets | Smooth seamless panels |
| Raygun gothic | Fins, domes, raygun flares | Capes, heroic poses | Engineering-overly-correct details |
| Dieselpunk | Streamlined wedges, ziggurats | Armored blocks, smokestacks | Rounded toy-like forms |
| Atompunk | Boomerangs, sputnik spheres | Tailfins, pinched-waist furniture | Military heaviness |
| Space-race brutalism | Rectilinear modules | Gantries, tube arrays | Decorative excess |
| Cassette futurism | Boxy consoles | CRT bulges, cable clusters | Holographic glass minimalism |
| Soviet cosmist retrofuture | Monumental axes | Orbital arcs, emblem blocks | Corporate lifestyle gloss |
| Retro anime mecha modernism | Angular humanoid mech frames | Dense urban infrastructure | Featureless megaforms |
| Y2K retrofuture pop | Blob ergonomics | Orb UI nodes, chrome bubbles | Matte utilitarian roughness |

## Density tuning

- **High-density styles:** cassette futurism, retro anime mecha, dieselpunk.
- **Medium-density styles:** Victorian steampunk, Soviet cosmist, space-race brutalism.
- **Low-to-medium density styles:** atompunk, raygun gothic, Y2K pop.

## Texture hierarchy

1. Macro texture: architecture and atmosphere.
2. Meso texture: paneling, trim, repeated motifs.
3. Micro texture: labels, fasteners, scratches, print marks.

For consistency, pick one hierarchy order and preserve it across scene elements.

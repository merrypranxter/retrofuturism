# Cross-domain Applications

## Architecture and interior design

- Use style entries to map facade language, structural systems, and interior object vocabulary.
- Keep timeline coherence (do not mix Victorian boiler logic with Y2K glossy plastics unless intentionally hybridized).

## Fashion and costume direction

- Tie silhouette to technology narrative:
  - Heavy utility harnessing for industrial styles.
  - Aerodynamic consumer polish for atompunk/Y2K.
  - Insignia-centered uniforms for state/mission styles.

## Graphic design and branding

- Use typography and symbol systems from each entry, not generic sci-fi fonts.
- Build icon libraries from motif vocabulary in `docs/motif_vocabulary.md`.

## Object and product design

- Start from material/manufacturing plausibility.
- Add ornament according to the style's ornament logic (functional, ideological, consumer, ritual).

## Illustration and concept art pipelines

1. Pick one core style + one adjacent modifier.
2. Lock palette and materials before detailing.
3. Apply composition template from `docs/composition_logic.md`.
4. Add micro-details: labels, fasteners, symbols, wear.
5. Run anti-pattern check before final render.
